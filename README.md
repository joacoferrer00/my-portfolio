# Joaquín Ferrer — Data Analyst Portfolio

Personal portfolio website showcasing data analysis projects, skills, and background.

## Tech Stack

- [React](https://react.dev/) + [Vite](https://vite.dev/)
- [Tailwind CSS](https://tailwindcss.com/)

## Local Development

```bash
npm install
npm run dev
```

Open [http://localhost:5173](http://localhost:5173) in your browser.

## Build

```bash
npm run build
```

Output goes to `dist/`. Ready to deploy on Vercel or Netlify as a static site.

## Content

All site content lives in the `/content` folder as markdown files:

- `content/about.md` — bio and background
- `content/skills.md` — skills by category
- `content/projects/*.md` — one file per project
