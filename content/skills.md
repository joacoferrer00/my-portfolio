# Skills

## Business Intelligence & Visualization
- **Power BI** — Advanced. DAX, Row-Level Security, performance optimization, enterprise-scale report design, data modeling for Service publishing.
- **Power Query (M)** — Data transformation, ETL logic, query folding awareness.
- **Dashboard design** — Custom themes, bookmarks, navigation, user-oriented layout for non-technical audiences.

## Data Modeling & SQL
- **SQL** — Advanced analytical queries, window functions, CTEs, query optimization for large datasets.
- **Data modeling** — Star schema, galaxy schema, dimensional modeling. Fact and dimension table design, surrogate keys, slowly changing dimensions.
- **Database platforms** — SQL Server, relational DBs, Elastic (API ingestion).

## Python & Analytics Automation
- **Python** — Pandas, NumPy, Matplotlib, Seaborn for EDA, data cleaning, and automation.
- **Automation** — Scheduled scripts, SMTP email automation, batch processing pipelines.
- **EDA** — Exploratory data analysis on large datasets (500k+ rows), outlier detection, distribution analysis, correlation analysis.

## Data Engineering & Pipelines
- **Apache Spark / PySpark** — Large-scale data transformation, DataFrame operations, schema management.
- **Microsoft Fabric** — Lakehouse architecture, OneLake, Spark notebooks, Delta Lake, SQL endpoints.
- **Delta Lake** — ACID transactions, schema evolution, time travel, upsert patterns.
- **Pipeline design** — Three-layer architecture (Raw → Staging → Curated), API pagination, Parquet/Delta storage.

## Automation & Orchestration
- **Power Automate** — Workflow automation, triggered pipelines, notifications.
- **Scheduled jobs** — Python-based scheduling, recurring data refresh orchestration.

## Collaboration & Tooling
- **Microsoft Fabric / Power Platform** — End-to-end BI ecosystem.
- **GitHub** — Version control for notebooks, DAX files, Python scripts.
- **Notion** — Project documentation and knowledge management.
- **Generative AI tools** — Used pragmatically to accelerate development, not as a crutch.
